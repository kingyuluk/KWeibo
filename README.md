## Overview

基于 Apple 与 Weibo 的 API 实现的第三方微博 App

## Funtions

* 用户登录，NSUserDefault 存储登录信息
* 主页微博展示
* 网络缓存的构建
* 微博图片自适应 cell 大小
* loadMore 控件
